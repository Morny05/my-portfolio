# Personal Portfolio

This is my personal portfolio website built with Next.js and Tailwind CSS. It showcases my background, technical skills, projects, certifications, and professional experience in DevOps, Cloud, and Big Data.

## Features

- Responsive modern design
- Smooth scrolling navigation
- About me section
- Experience timeline
- Skills showcase
- Certifications
- Projects
- Contact section

## Tech Stack

- Next.js
- React
- Tailwind CSS
